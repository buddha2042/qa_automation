## SisenseDashboard testion App

React/Next.js front end and backend.

### Prerequisites
- Node 18+
- Python 3.10+

### One-time setup (Node + Python)

Run:

```bash
npm run setup
```

This command will:
- install Node dependencies (`npm install`)
- create a local Python virtual environment at `.venv` (if missing)
- install Python requirements from `requirements.txt`

### Run the app

```bash
npm run dev
```

Open http://localhost:3000

### Python dependencies used by `.smodel` compare

Defined in `requirements.txt`:
- `pandas`
- `openpyxl`
